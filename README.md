
# Data Analytics Project

# customer_behaviour_analysis

## Overview
This project demonstrates an end-to-end data analytics workflow, starting from raw data loading to final business reporting. The main goal was to analyze the dataset, clean and transform the data, extract insights using SQL, and present the findings through a Power BI dashboard, a report, and a presentation.

This project highlights practical skills in Python, SQL, data cleaning, exploratory data analysis (EDA), dashboard development, and business communication.

## Dataset
The dataset used in this project contains structured business data for analysis.

### Dataset tasks:
- Loaded dataset in Python
- Checked data types and structure
- Identified missing values and duplicates
- Cleaned and standardized columns
- Prepared data for SQL analysis and dashboarding

## Tools and Technologies
- **Python** – data loading, cleaning, and EDA
- **Pandas, NumPy, Matplotlib, Seaborn** – data analysis and visualization
- **PostgreSQL / MySQL / SQL Server** – SQL querying and analysis
- **Power BI** – dashboard creation
- **Gamma** – presentation creation
- **Microsoft Word / PDF** – report writing

## Project Steps

### 1. Data Loading
- Imported the dataset into Python
- Reviewed rows, columns, and data types
- Checked overall data quality

### 2. Exploratory Data Analysis (EDA)
- Generated summary statistics
- Studied patterns, trends, and distributions
- Created charts to understand the data better
- Identified anomalies and useful insights

### 3. Data Cleaning
- Handled missing values
- Removed duplicates
- Fixed inconsistent formats
- Standardized column names and values
- Prepared clean data for analysis

### 4. SQL Analysis
- Loaded cleaned data into relational databases
- Wrote SQL queries in PostgreSQL, MySQL, and SQL Server
- Performed filtering, grouping, joins, aggregations, and KPI analysis

### 5. Dashboard Development
- Imported final data into Power BI
- Built an interactive dashboard with important KPIs and charts
- Added filters and slicers for easier analysis
- Focused on simple and business-friendly visuals

### 6. Reporting and Presentation
- Created a report summarizing the process and findings
- Built a presentation using Gamma
- Highlighted important insights and recommendations

## Dashboard
The Power BI dashboard was created to present key insights in a clear and interactive way.

### Dashboard highlights:
- KPI cards
- Trend analysis charts
- Category comparisons
- Filters and slicers
- Clear visual storytelling

## Results
This project helped generate useful insights through a structured analytics process.

### Key outcomes:
- Better understanding of the dataset through EDA
- Clean and reliable data for analysis
- Insights extracted using SQL queries
- Interactive Power BI dashboard for decision-making
- Professional report and presentation for communication
